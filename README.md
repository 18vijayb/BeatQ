## Running Locally 
(i) pip3 install virtualenv
(2) virtualenv BeatQ
(3) source BeatQ/Scripts/activate  or source BeatQ/bin/activate
(4) pip3 install -r requirements.txt
(5) set FLASK_APP=flaskr
(6) set FLASK_ENV=dev
(7) flask run
